## üìñ About The Project

Velaro is a premium, fully responsive front-end e-commerce web application tailored specifically for streetwear and modern fashion brands. Built with an emphasis on brutalist yet refined aesthetics, the platform delivers a highly engaging and immersive shopping experience.

It features stunning full-height hero sliders, smooth micro-animations, interactive quick-view product modals, and a completely custom-built slide progress interface inspired by top-tier Framer templates.

## ‚ú® Key Features

- **Immersive Hero Slider**: Custom-built, full-width automated hero slider with bespoke progress tracking, opacity transitions, and dual-gradient overlays.
- **Fluid & Responsive Design**: Meticulously crafted responsive layouts that adapt seamlessly from deep mobile screens up to ultra-wide displays.
- **Interactive Quick View**: High-converting product modal featuring a selectable thumbnail gallery, interactive size and color pickers, and live price/cart integration.
- **Advanced State Management**: Real-time shopping cart handling addition, deletion, quantity adjustments, and subtotal calculation directly in the UI.
- **Infinite Scrolling Marquees**: "The Streets Are Talking" review section utilizing CSS keyframe animations for an infinite, bidirectional scrolling marquee.
- **Modern Typography**: Bold, aggressive headers paired with highly legible body text.

## üöÄ Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

* npm (v8 or higher)
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/your-username/velaro-ecommerce.git
   ```
2. Navigate to the project directory
   ```sh
   cd velaro-ecommerce
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Start the development server
   ```sh
   npm run dev
   ```

## üé® Design System

**Typography:**
- Headers & Accents: `Big Shoulders Display` (Uppercase, tight tracking)
- Body text: `Poppins` (Readable, medium line-heights)

**Color Palette:**
- Primary: `#111111` / `#1a1a1a` (Deep blacks and charcoal grays)
- Backgrounds: `#ffffff` / `#f5f5f5` / `#e5e5e5` (Clean whites and soft off-whites)
- Accents: High contrast whites on dark backgrounds with subtle transparency (`rgba(255,255,255,0.2)` borders).
